## Assumptions of multiple linear regression (MLR) analysis If we want to perform MLR analysis, the following must hold true: 1) Wages (\(Y\)) must be a linear response to the covariates (\(X\)), education, age, sex and language. \(\mathb{Y} = \mathb{X}\beta + \epsilon\). 2) The error needs to have equal and uncorrelated variance for each covariate. \(Cov{\epsilon} = \sigma^2 \mathb{I}\). 3) Errors need to be additive (see formula related to pt. 1). 4) Errors need to be normally distributed. \(\epsilon ~ N_n(\mathb{0}, \sigma^2 \mathb{I})\).
